# 🌱 OptiCrop – Smart Agricultural Production Optimization Engine

## 📌 Project Overview
OptiCrop is a Machine Learning-based crop recommendation system that predicts the most suitable crop using soil nutrients and environmental conditions. The application helps farmers make better crop selection decisions by analyzing agricultural parameters such as Nitrogen, Phosphorous, Potassium, Temperature, Humidity, pH, and Rainfall.

---

## 🎯 Problem Statement
Selecting the right crop based on soil and weather conditions is difficult for many farmers. Incorrect crop selection can reduce productivity and increase losses. OptiCrop solves this problem by recommending the most suitable crop using a trained K-Nearest Neighbors (KNN) Machine Learning model.

## ⚙️ Installation

1. Clone the repository

```bash
git clone <repository-link>
```

2. Navigate to the project folder

```bash
cd OptiCrop
```

3. Install the required packages

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

---

## 🚀 Usage

1. Open the application in your browser.
2. Navigate to the **Find Your Crop** page.
3. Enter the required agricultural parameters.
4. Click **Predict Crop**.
5. View the recommended crop.
